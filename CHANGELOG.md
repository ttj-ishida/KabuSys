## [0.1.0] - 2026-03-15
初回リリース。日本株自動売買システムのパッケージ骨組みを追加しました。

### 追加
- 基本パッケージ `kabusys` を追加
  - パッケージ説明ドキュメンテーション文字列: "KabuSys - 日本株自動売買システム"
  - バージョン識別子 `__version__ = "0.1.0"` を設定
  - パブリック API としてエクスポートするサブパッケージを `__all__ = ["data", "strategy", "execution", "monitoring"]` にて明示
- サブパッケージのスケルトンを追加
  - `kabusys.data` - データ取得・管理（プレースホルダ）
  - `kabusys.strategy` - 売買戦略定義（プレースホルダ）
  - `kabusys.execution` - 注文執行・ブローカー連携（プレースホルダ）
  - `kabusys.monitoring` - 監視・ログ・状態管理（プレースホルダ）
- 各サブパッケージに対し、空の `__init__.py` を配置してパッケージ構造を確立
