## [0.1.0] - 2026-03-15

初回リリース。パッケージの基本的なスキャフォールドを追加しました。

### Added
- 新規パッケージ `kabusys` を追加。
  - パッケージ説明: "KabuSys - 日本株自動売買システム"（トップレベル docstring）。
  - バージョン情報: `__version__ = "0.1.0"`.
  - 公開モジュール一覧: `__all__ = ["data", "strategy", "execution", "monitoring"]` を定義し、外部からのインポートエントリポイントを明示。
- 基本サブパッケージ（空のモジュールファイルを含む）を追加:
  - `kabusys.data` — 市場データ取得／管理を想定するモジュール（スキャフォールド）。
  - `kabusys.strategy` — 売買戦略（シグナル生成等）を実装するためのモジュール（スキャフォールド）。
  - `kabusys.execution` — 注文執行・約定処理を担うモジュール（スキャフォールド）。
  - `kabusys.monitoring` — 監視・ログ・アラート等を担うモジュール（スキャフォールド）。

## 参考・補足
- 現状はパッケージ構造（モジュールの骨組み）とバージョン情報のみが実装されています。各サブパッケージの具体的な機能（データ取得ロジック、戦略アルゴリズム、注文送信、監視機能など）は今後の実装対象です。
- 利用者は `from kabusys import data, strategy, execution, monitoring` の形式で各サブパッケージへアクセスできます。
- 次のリリースでは各サブパッケージ内の具体的な API 、依存関係、使用例、互換性情報を記載予定です。
