0.1.0 - 2026-03-15
-----------------

Added
- 初期リリース。
- 新規パッケージ "kabusys" を追加（パッケージ docstring に「日本株自動売買システム」と明記）。
- バージョン定義を追加: `__version__ = "0.1.0"`。
- パッケージ公開 API を明示: `__all__ = ["data", "strategy", "execution", "monitoring"]` を設定。
- 主要サブパッケージ（スケルトン）を追加:
  - `kabusys.data` — データ取得・管理用モジュール（現状は空の初期化ファイル）。
  - `kabusys.strategy` — 売買戦略定義用モジュール（現状は空の初期化ファイル）。
  - `kabusys.execution` — 注文実行/ブローカー連携用モジュール（現状は空の初期化ファイル）。
  - `kabusys.monitoring` — 監視・ロギング/稼働監視用モジュール（現状は空の初期化ファイル）。
- パッケージ構成は基本的なモジュール分割（data/strategy/execution/monitoring）を想定したスケルトン実装。

Notes / 今後の予定
- 現時点では各サブパッケージは初期化ファイルのみのスケルトン状態であり、具体的な機能実装（API 実装、データ取得ロジック、戦略アルゴリズム、注文処理、監視機能など）は未実装です。
- 今後のリリースで以下を追加予定:
  - 各モジュールの具体的実装（データ取得、バックテスト、実取引実行等）
  - 単体テスト・統合テストおよび CI 設定
  - 利用方法ドキュメント（README / API ドキュメント）
  - セキュリティ・エラーハンドリングの強化
