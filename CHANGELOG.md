## [0.1.0] - 2026-03-15

### 追加
- 初期リリースとしてプロジェクトのパッケージ骨格を追加。
  - パッケージ名: `kabusys`（モジュール・ドキュメンテーション文字列: "KabuSys - 日本株自動売買システム"）。
  - バージョン情報: `__version__ = "0.1.0"` をルートパッケージに定義。
  - 公開 API: `__all__ = ["data", "strategy", "execution", "monitoring"]` を設定。
  - サブパッケージ雛形を作成:
    - `kabusys/data` — データ取得・管理用モジュールのためのプレースホルダ。
    - `kabusys/strategy` — 売買戦略実装用のプレースホルダ。
    - `kabusys/execution` — 注文送信・取引実行用のプレースホルダ。
    - `kabusys/monitoring` — 監視・ログ・メトリクス用のプレースホルダ。
- ディレクトリ構成（主要ファイル）
  - `src/kabusys/__init__.py`（パッケージ宣言、バージョン、__all__）
  - 各サブパッケージに空の `__init__.py`（将来的な実装のための雛形）

### 注意事項 / マイグレーション
- 本リリースはあくまでプロジェクトの骨格（スケルトン）であり、各サブパッケージ内に具体的な実装は含まれていません。今後、各モジュールに機能を追加していく予定です。
- 利用開始例:
  - importによるバージョン確認: `import kabusys; print(kabusys.__version__)`
  - パッケージ API の参照は `kabusys.data`, `kabusys.strategy`, `kabusys.execution`, `kabusys.monitoring` を想定。

### 既知の制限
- 実装未完のため、現時点での動作は限定的です（機能はほぼ存在しません）。ビルドやインポートの検証のみ可能。
