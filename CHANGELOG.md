## [0.1.0] - 2026-03-15

### Added
- 初期リリースを作成しました。
  - パッケージ名: `kabusys`（日本株自動売買システムを想定したベースパッケージ）
  - ルートモジュール `src/kabusys/__init__.py` を追加。
    - パッケージのバージョン情報 `__version__ = "0.1.0"` を定義。
    - 公開 API を示す `__all__ = ["data", "strategy", "execution", "monitoring"]` を定義。
    - パッケージの簡単な説明文（モジュールドックストリング）を追加。
  - 機能別サブパッケージの雛形を追加（空の __init__.py を含む）
    - `src/kabusys/data` : データ取得・管理に関するモジュール用の名前空間
    - `src/kabusys/strategy` : 売買戦略ロジック用の名前空間
    - `src/kabusys/execution` : 注文実行（ブローカー等とのやり取り）用の名前空間
    - `src/kabusys/monitoring` : 稼働監視・ログ・アラート用の名前空間
- パッケージ構造の基本骨格を確立しました。今後各サブパッケージに具体的な実装（関数・クラス・API）が追加される予定です。

### Notes
- 現時点では各サブパッケージは初期雛形（空の __init__.py）であり、公開 API は `__all__` により上記4つの名前空間が示されています。
- 次のリリースでは、各領域（データ取得、戦略、注文実行、監視）に対する具体的な実装とテストを追加する予定です。
