# AIドキュメント生成 プロンプト

あなたは技術ドキュメントの専門家です。
以下のコードベースのドキュメントを日本語で作成してください。

## README.md の構成

1. **プロジェクト概要** - 何をするシステムか
2. **主要機能** - 機能一覧（箇条書き）
3. **セットアップ手順** - 環境構築の手順
4. **使い方** - 基本的な使用例
5. **ディレクトリ構成** - ファイル構成の説明
6. **設定** - 設定ファイルの説明
7. **ライセンス**

## CHANGELOG.md の構成

Keep a Changelog（https://keepachangelog.com）準拠:

```markdown
# Changelog

## [Unreleased]
### Added
### Changed
### Fixed
```

## 注意事項

- 技術的に正確な内容を記載する
- コードから読み取れる情報のみを記載する
- 推測で不正確な情報を書かない
